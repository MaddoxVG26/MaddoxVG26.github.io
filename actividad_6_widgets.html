Siglas

OSI

Modelo conceptual que divide las funciones
IP

Protocolo que identifica dispositivos en una red
TCP

protocolo de control de transmision, garantiza la entrega correcta de los datos en una red
UDP

protocolo de datagramas de usuario, permite el envio de mensajes sin necesidad de establecer una coneccion
SSH

protocolo que permite el acceso seguro a un dispositivo remoto a traves de una red

FTP

protocolo de transferencia de archivos, utilizado para transferir archivos entre dispositivos en una red

SMTP

protocolo simple de transferencia de correo, utilizado para enviar correos electronicos entre servidores

POP

protocolo de oficina de correos, utilizado para recuperar correos electronicos desde un servidor

DHCP

protocolo de configuracion dinamica de hosts, asigna automaticamente direcciones IP a los dispositivos de una red

ONS

sistema de nombres de dominio, traduce nombres de dominio en direcciones IP
